TechVault - Modern E-commerce Platform for Tech Accessories
Project Overview
TechVault is a modern, responsive e-commerce platform built with React and Tailwind CSS, specializing in tech accessories including headphones, chargers, cables, power banks, and other gadgets. The platform delivers a premium shopping experience with a focus on clean design, smooth animations, and optimal performance.

Tech Stack
Core Technologies
React 19 - UI framework with hooks and functional components

TypeScript 6.0 - Type-safe development

Vite 8.0 - Build tool and development server

Tailwind CSS 4.3 - Utility-first styling

React Router DOM 7.17 - Client-side routing

State Management & Data
React Context API - Cart, Wishlist, and Theme management

React Query (TanStack Query) - Server state management and data fetching

Axios - HTTP client for API requests

UI Components & Styling
Lucide React - Icon library

React Icons - Additional icon sets

Slick Carousel - Product carousels with responsive support

Sonner - Toast notifications

Development Tools
ESLint - Code linting and formatting

PostCSS - CSS processing with Autoprefixer

TypeScript ESLint - TypeScript linting

Component Architecture
Presentational Components - Focus on UI rendering, receive data via props

Container Components - Handle logic, state, and data fetching

Page Components - Compose presentational and container components

Styling Approach
Tailwind-First: Use Tailwind utility classes for all styling

Custom Classes: Extract repeated patterns into custom utility classes

Responsive Design: Mobile-first approach using Tailwind's responsive modifiers

Dark Mode: Support dark/light themes using CSS variables
